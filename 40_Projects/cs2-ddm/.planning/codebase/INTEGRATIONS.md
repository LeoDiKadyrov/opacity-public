_Generated: 2026-04-30_

## Summary

This project has no cloud service integrations or API calls. All external dependencies are local: CS2 demo files read from disk, `.tri` mesh files for BVH geometry, and output written to a local CSV file. The only "external" surface is the demoparser2/awpy toolchain which reads proprietary Valve demo format.

## Data Sources

**CS2 GOTV Demo Files (`.dem`):**
- Format: Valve Source Engine demo format (binary, proprietary)
- Location: `D:\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\` (hardcoded in `run_analysis.py` and `kill_rate_analysis.py`)
- Input directory for pipeline: `for_analysis/` (drop files here for `/add-demo` skill)
- Parser: `demoparser2` — Rust extension wrapping CS2's demo protocol
- Events extracted: `player_hurt`, `weapon_fire`, `player_death`, `smokegrenade_detonate`, `smokegrenade_expired`, `player_blind`, `round_start`
- Per-tick props extracted: player X/Y/Z, yaw/pitch, velocity, team_num, is_alive, usercmd_viewangle, aim_punch_angle, crouch state

**CS2 Map Geometry (`.tri` files):**
- Format: proprietary awpy triangle mesh format (BVH precomputed)
- Location: `C:\Users\Leo\.awpy\tris\` — resolved via `awpy.data.TRIS_DIR`
- Download method: `awpy_cli(['get', 'tris'])` (one-time setup)
- Used by: `t0_detector.py` — `VisibilityChecker(path=Path(TRIS_DIR) / '{map_name}.tri')`
- Purpose: Möller-Trumbore ray-triangle intersection for geometric LOS (T0 detection)
- Maps confirmed working: `de_ancient`, `de_mirage`

## Libraries Acting as External Integrations

**demoparser2 (Rust extension):**
- Role: sole interface to CS2 demo binary format
- API surface used:
  - `DemoParser(path).parse_ticks(props, ticks=None)` → per-tick DataFrame
  - `DemoParser(path).parse_events(event_name, props)` → event DataFrame
  - `DemoParser(path).parse_header()` → dict with `map_name`, `demo_type`
- Version sensitivity: 0.41.2 required for CS2 patch 14155+ demos; older versions silently fail on new demos
- No network calls — reads local file only

**awpy VisibilityChecker:**
- Role: BVH geometric line-of-sight oracle
- API surface used in `t0_detector.py`:
  ```python
  checker = VisibilityChecker(path=Path(TRIS_DIR) / 'de_ancient.tri')
  checker.is_visible(start_tuple, end_tuple) -> bool
  ```
- Rays cast to 8 AABB corners of enemy bounding box (32x32x72 standing, 32x32x54 crouching)
- Eye height offset: +64u (standing) / +46u (crouching) above demoparser2 Z coordinate
- Smoke suppression: sphere intersection test (radius 144u) against active smokes before ray cast
- Flash suppression: `parse_flash_intervals()` returns `List[Tuple[int,int]]` checked per-tick

## File I/O

**Input:**
- `*.dem` files — CS2 demo binaries, read by demoparser2

**Output:**
- `cs2_engagement_analysis_results.csv` — persistent results DB; append+dedup by `match_id` via `csv_utils.save_results()`; write-protected by Claude hook
- `{player}_attempts.csv` — T0-anchored duel attempt records written by `kill_rate_analysis.py`
- `analysis_plots/` — chart PNGs generated by `visualize_results.py`
- `ddm_analysis.log` — pipeline execution log
