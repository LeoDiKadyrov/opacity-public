# Phase 09.1 Plan 04: Per-Steamid DataFrame Cache (SC4) Summary

One-liner: Per-steamid DataFrame cache built once in `analyze_demo()` via `groupby('steamid', sort=False)` and threaded through `analyze_engagement_episode → _resolve_t0 → find_t0`, replacing repeated bool-filters on `all_ticks_df` with O(1) dict lookups while preserving the bool-filter as a fallback.

## What Was Done

### Task 1 — `find_t0` kwarg + dual-branch (commit `71a3dfc`)
- Added `from typing import Dict` to t0_detector.py.
- Extended `find_t0` signature with `ticks_by_sid: Optional[Dict[int, pd.DataFrame]] = None` as the LAST kwarg (preserves existing positional + keyword call shapes).
- Replaced the two bool-filter blocks (player_steamid → `p_df`, enemy_steamid → `e_df`) with the dual-branch pattern from RESEARCH.md §Pattern 3:
  - Cache hit (`ticks_by_sid is not None and sid in ticks_by_sid`) → take cached frame, apply tick-range filter only.
  - Cache miss / `None` → original bool-filter on `all_ticks_df` (D-04 zero-regression contract).
- Sort already done at cache build time; cached branch skips `.sort_values("tick")` for speed.
- All other find_t0 logic (BVH raycast, smoke check, flash check, AABB iteration) untouched.

### Task 2 — Cache build + thread-through (commit `b9b3be0`)
- Inside `analyze_demo`, after the `steamid` coerce-to-int64 block, added the cache build:
  ```python
  ticks_by_sid: Dict[int, pd.DataFrame] = {
      int(sid): g.sort_values("tick")
      for sid, g in all_ticks_df.groupby("steamid", sort=False)
  }
  ```
- `analyze_engagement_episode` signature gained `ticks_by_sid: Optional[Dict[int, pd.DataFrame]] = None`. Forwards to `_resolve_t0`.
- `_resolve_t0` signature gained the same kwarg. Forwards to `self.t0_detector.find_t0(...)`.
- `analyze_demo` per-moment loop passes `ticks_by_sid=ticks_by_sid` to `analyze_engagement_episode`.
- `groupby` count audit: 1 occurrence in ddm_analyzer.py — the build site only. Cache is not defeated by re-grouping inside the loop.
- `ticks_by_sid` count: 6 occurrences in ddm_analyzer.py (build + signatures + forwards).
