**Pattern B — `_parse_ticks_maybe_selective` helper (SC3)**, replaces lines 731-736:
```python
def _parse_ticks_maybe_selective(
    self,
    props: List[str],
    anchor_ticks: Optional[List[int]],
) -> pd.DataFrame:
    """Selective parse if flag enabled and anchors present, else full parse.
    Pitfall guard: empty union list silently returns full parse — explicit guard required.
    """
    from config import PARSE_TICKS_SELECTIVE
    if PARSE_TICKS_SELECTIVE and anchor_ticks:
        window = self._build_tick_window_union(anchor_ticks)
        if len(window) > 0:  # CRITICAL GUARD - ticks=[] returns FULL parse silently
            try:
                return pd.DataFrame(self.parser.parse_ticks(props, ticks=window))
            except Exception as e:
                self.logger.warning(f"Selective parse_ticks failed, falling back to full: {e}")
    try:
        return pd.DataFrame(self.parser.parse_ticks(props))
    except Exception as e:
        self.logger.warning(f"Optional tick props unavailable, falling back to required only: {e}")
        return pd.DataFrame(self.parser.parse_ticks(self._TICKS_REQUIRED))
```
**Critical ordering:** parse_events MUST run BEFORE `_parse_ticks_maybe_selective` because anchor_ticks comes from `all_hurt_df`. The current order (ticks first, events second) MUST be reversed.

**Pattern C — `_build_tick_window_union` helper (SC3)**, new method:
```python
import numpy as np

_SELECTIVE_WINDOW_BEFORE_TICKS: int = 384   # set in config.py
_SELECTIVE_WINDOW_AFTER_TICKS:  int = 192

def _build_tick_window_union(self, anchor_ticks: List[int]) -> List[int]:
    if not anchor_ticks:
        return []
    arrs = [
        np.arange(t - _SELECTIVE_WINDOW_BEFORE_TICKS,
                  t + _SELECTIVE_WINDOW_AFTER_TICKS,
                  dtype=np.int32)
        for t in anchor_ticks
    ]
    union = np.unique(np.concatenate(arrs))
    union = union[union >= 0]
    return union.tolist()
```
