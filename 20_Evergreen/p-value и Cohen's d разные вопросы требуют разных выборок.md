# p-value и Cohen's d: разные вопросы требуют разных выборок

Есть два разных вопроса, которые люди часто путают. Первый: есть ли эффект вообще? Это p-value. Второй: насколько сильный этот эффект? Это d (Cohen's d). Большинство лабораторных исследований отвечают только на первый вопрос — и то кое-как.

Стандартная лабораторная выборка — n=20 на группу. Этого не хватает даже чтобы надёжно зафиксировать сам факт эффекта. Для нормальной статистической мощности нужно n=64. А чтобы получить достаточно узкий доверительный интервал для размера эффекта — нужно уже 3000+. Разница в двести раз. Вот почему лаборатории физически не могут это делать: дорого.

Simonsohn показывает конкретно: даже при n=205 на группу (что уже очень много) доверительный интервал для d растягивается от 0.3 до 0.7 — то есть ты не знаешь, эффект маленький или большой. Исследование «почти всегда» даст значимый результат, но о силе эффекта оно не скажет ничего полезного.

Практический вывод: если делаешь свой ресёрч и хочешь не просто «есть эффект / нет эффекта», а что-то реальное про его величину — нужно сначала определить SESOI (smallest effect size of interest), провести пилот, рассчитать нужную выборку. Иначе ты потратишь время на данные, которые ничего не докажут про силу эффекта.

Для маленьких независимых исследований из этого есть выход: мета-анализ. Одно исследование на 200 человек не даёт размер эффекта, но десять таких исследований вместе — уже дают. Отсюда и сдвиг в психологии: маленькие работы теперь воспринимаются как кирпичи для мета-анализа, а не самостоятельные открытия.
