**Title:** `Режимы Лазаря как граф — BASIC ID, узел Ярцева, место в Психодемии`

---

Арнольд Лазарь в 1973 ввёл мультимодальную терапию через акроним **BASIC ID** — семь модальностей: Behavior, Affect, Sensation, Imagery, Cognition, Interpersonal, Drugs/biology. Это не школа и не теория мотивации — это диагностическая сетка плюс правило сцепления: интервенция в одной модальности тянет смежные. Лазарь утверждал, что без покрытия всех семи терапия упускает рецидивные узлы.

Граф появляется через две операции, обе формализованы у Лазаря:

1. **Modality Profile** — клиента картируют по семи модальностям отдельно (что делает, что чувствует, ощущает, воображает, думает, с кем, что в теле/химии).
2. **Bridging и Tracking** — прослеживается каузальная цепочка между модальностями у конкретного клиента: когнитивный триггер → телесное ощущение → образ → поведение. Узлы = модальности, рёбра = эмпирически наблюдаемые переходы. Это и есть concept graph в исходной форме.

**Александр Ярцев в MentalTech Lab.** Честно: его конкретная формализация BASIC ID как графа из верифицируемых публичных источников до cutoff мне неизвестна. Три правдоподобные интерпретации, согласующиеся с профилем MentalTech Lab (доказательная психология, оцифровка протоколов, Maria Данина cofounder):

- **Граф предикторов** — узлы = модальности, ребро = условная вероятность снижения симптома в B при интервенции в A. Требует датасета сессий.
- **Граф интервенций** — узлы = техники по модальностям (когнитивная реструктуризация для C, экспозиция для B, релаксация для S), рёбра = совместимость/последовательность.
- **Граф концепций для обучения** — узлы = модальности и подкомпоненты, рёбра = "необходим для понимания". EdTech-сценарий, прямой fit с Психодемией.

Без primary source (статья, репозиторий, лекция Ярцева) выбор — спекуляция. Если граф существует, он либо в академическом контенте MentalTech Lab, либо в учебной программе Психодемии/Балансити.

**Место в Психодемии.** Если модуль "Мультимодальная терапия Лазаря" существует как отдельный — типичная подача в EdTech по психологии:

- Историческая надстройка над behaviorism (Лазарь — ученик Уолпе, отошёл от чистого бихевиоризма).
- Инструмент диагностики, не самостоятельный метод. Студент строит Modality Profile, но интервенции берёт из КПТ, схема-терапии, ОРКТ — школ, которые в Психодемии раскрыты как полноценные курсы.
- Мост между школами. Здесь граф работает как онтология сравнения: BASIC ID позволяет сопоставлять интервенции разных школ по покрытию модальностей.

**Brutal honest.** Мультимодальная терапия Лазаря — зрелая мета-рамка 1970–80-х, не современный фронтир. По effect size в RCT проигрывает протокольному CBT, отдельных крупных meta-analyses под её именем нет — её сила в концептуализации, не в outcome. Если Ярцев строит граф ради операционализации (датасет, ML-классификация случаев) — это новая ценность. Если ради педагогики — это переупаковка зрелого материала с UX-надстройкой.
