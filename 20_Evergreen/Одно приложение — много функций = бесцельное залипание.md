# Одно приложение — много функций = бесцельное залипание

Когда одно приложение закрывает всё — музыку, общение, новости, заметки, фото — оно перестаёт ассоциироваться с чем-то конкретным. Нет чёткой цели захода, значит, мозг не знает, зачем туда идёт. А раз не знает — идёт просто так, по привычке, в любой непонятной ситуации.

Это не проблема силы воли. Это проблема архитектуры ассоциаций. Телефон берёшь — рука сама открывает Telegram, потому что там всё. Скука? Telegram. Пауза? Telegram. Неловкость? Telegram. Приложение стало дефолтным ответом на любое состояние неопределённости.

Решение — дробление по функциям. Музыка в одном месте, новости в другом, заметки в третьем. Каждый инструмент с одной ролью = чёткая ассоциация = осознанный заход. Зашёл в музыкалку — слушаешь музыку. Зашёл в мессенджер — общаешься. Нет путаницы, нет бесцельного серфинга внутри одного монстра.

Принцип работает не только для приложений. Рабочий стол, вкладки браузера, папки — везде, где одно место служит всем целям сразу, теряется осознанность. Структура создаёт намерение.
