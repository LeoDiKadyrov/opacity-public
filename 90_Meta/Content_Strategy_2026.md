# CONTENT STRATEGY 2026 — узкий голос вместо широкой аудитории

> Аудитория собирается вокруг конкретного **ремесла**, не вокруг "личного бренда".

---

## 0. СТРУКТУРНЫЙ КОНФЛИКТ

"Набирать аудиторию" конфликтует с DNA. 4 года Telegram-канал = анти-perform, 80–130 подписчиков, дневник для себя. Voice Profile §8 "Polish as a Lie", §9 "Audience = peer syncing, не teaching".

**Решение:** не выкручивать ручку громкости. Поставить натуральный голос туда, где он = редкость.

**3 SHARP темы** где голос = редкость: ЦР-инсайдер · interconnectedness-практик · AI-tooling-builder.

## 3. ТРИ СТАВКИ

### Ставка 1 — LinkedIn (RU, fintech-инсайдер) — главная

**Тема:** ремесло Implementation Manager Цифрового рубля. Не сам ЦР (NDA). **Мета-наблюдения**: реакция банков, провалы коммуникации, монки-ворк vs реальные решения, рынок имплементации.

**Почему DNA-совместимо:**
- Voice Profile §9: LinkedIn = "Tool. Formal protocol, but still grounded"
- Brutal honesty про индустрию = редкое на LinkedIn
- "80% banking work = monkey work" (Fact Dossier §7) — готовый питч
- NDA = forcing function на мета-уровень

**Ритм:** 1 пост / неделя × 12 мес = 50 постов. Цель — репутация в узком круге, не виральность. **500 инсайдеров > 50K туристов.**

**Формат:** 600–1000 слов, аудит-список Win/Loss, без expert posturing.

**Запретное:** имена банков, конкретные сделки, тех. детали ЦР.

### Ставка 2 — X (английский, AI-tooling builder)

**Тема:** Claude Code плагины (health-agent, psycho-plugin, wtp-plugin), MCP-серверы, second-brain pipeline, briefing.py. **Пользователь, строящий инструменты для себя**, не разработчик.

**Почему DNA-совместимо:**
- Английский = raw mode (Voice Profile §2)
- "Я блоха" фрейм = build-in-public натурально
- Антицель "не наёмный разработчик" не нарушается — vibe-coding для своих

**Ритм:** 2-3 поста / неделя — скрин/видео + 1 предложение. 1 раз / месяц — длинный thread про плагин.

**Формат:** show, don't tell. Скриншот → "this saved me X" → github link.

### Ставка 3 — Стрим kick.com/leodikadyrov (уже работает)

Не трогать частоту. **Один UX-апгрейд:** хайлайты 30-60 сек → TikTok/Shorts/Reels. Не для роста стрима — для voice samples в дикую аудиторию.

Антицель: не превращать в работу. Если стрим начинает ощущаться как работа — режь.

## 4. ЧТО НЕ ДЕЛАТЬ

- **Не запускать YouTube long-form.** Требует perform + edit, противоречит "Abrupt Exit".
- **Не объединять каналы под один бренд.** Канал-дневник, LinkedIn-инсайдер, X-builder = три разных voice mode.
- **Не нанимать ghostwriter / SMM.** Polish as a Lie.
- **Не ставить метрики подписчиков как primary.** Primary метрика = **% постов которые ты бы написал сам в дневник**. Падает → не работает.
- **Не строить "набор аудитории" книгу №2.** Только с дистрибуцией и нишей.

## 6. ЧЕКПОИНТЫ

**6 мес (ноябрь 2026):**
- LinkedIn: 25 постов, 300+ подписчиков, 3+ входящих от HR/банкиров/fintech
- X: 50+ постов, 1 плагин с 10+ внешними юзерами
- Канал не изменился = win, не fail

**12 мес (май 2027):**
- LinkedIn: 500+ inside-аудитория, оффер консалтинга / выступления
- X: 1000+ followers в нише MCP/Claude Code, github stars >100
- Решение по DDM (живёт / закрыто)

**Stop-rule:** 6 мес → LinkedIn не двигается = тема не та. Меняй угол, не тон.

## 8. ГЛАВНЫЙ ПРИНЦИП

Аудитория собирается **вокруг ремесла**, которое ты делаешь и так. Не вокруг Арыстана-как-личности. Вокруг **ЦР-инсайдера** и **vibe-coder-builder**.

Если контент перестаёт быть тем, что ты бы и так делал — стратегия ломается. Откатывай, не дави.
